Lab 10 Problem 3: Flask Web Page

How I installed Flask and built this page.

Installation Process

I already had Python installed, so I installed Flask by running pip install flask in the terminal.

Building the Page

I created a Python script main.py and an HTML template file /templates/index.html to display content on the web page.

Difficulties Encountered

The main difficulty I encountered was understanding how Flask renders templates and sets up routes. I solved this by consulting Flask documentation and reading through tutorials on their official website.

Flask requires a specific folder structure, such as placing HTML templates in the /templates directory. Initially, I tried rendering a file from the root directory, which resulted in an error. I read Flask's documentation, which clarified the need to place all templates in the /templates folder. I reorganized my files accordingly.

This is what my main.py file looks like:


from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run()
